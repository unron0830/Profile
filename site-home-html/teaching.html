可開設課程

核心課程

  • 地理資訊系統概論（Introduction to Geographic Information Systems）
    介紹向量與網格資料模型、空間分析基礎與地理資料管理概念， 說明 GIS 於環境監測、工程分析與資源管理之應用， 並搭配 QGIS / ArcGIS 軟體進行基礎實作。
  • 遙測概論（Introduction to Remote Sensing）
    說明衛星遙測基本原理、電磁波與地物光譜特性， 包含影像前處理、常用指標分析（如 NDVI）， 並結合災害監測與環境變遷之實際案例進行說明。
  • 地質學工程應用（Engineering Geology Applications）
    探討地質條件於工程規劃與邊坡穩定分析中的角色， 結合崩塌案例、地形資料與監測成果， 說明地質資訊於工程安全與風險評估之應用。

進階與應用課程

  • 網路地理資訊系統（WebGIS）
    使用ASP.NET MVC C# 建置網頁並以IIS進行發布 使用openlays.js 架設網路地圖模組 實作SQL 資料庫設計 以PostgreSQL / GeoServer 發布WMS / WFS / WMTS 地圖服務 前後端整合與資料視覺化於決策支援系統中的應用 製作調查APP或資訊整合APP，如口罩地圖、髒亂通報APP
  • 雲端地理資訊系統與遙測分析（Cloud-based GIS & Remote Sensing）
    以 Google Earth Engine 與 Python 為核心工具， 介紹大尺度遙測資料之雲端處理流程與時間序列分析方法， 並說明常用機器學習演算法於遙測分析中的應用， 包含特徵選取、模型訓練與驗證流程， 探討雲端運算於環境監測與災害分析之實務應用。
  • 數值地形資料處理與分析（DEM Analysis）
    介紹數值地形資料（DEM）之來源與特性， 說明坡度、流向、集水與 DoD（DEM of Difference）等地形衍生因子計算， 並應用於崩塌分析與地形變遷評估。

技術、實作與教學導向課程

  • 地理與環境資料之程式化處理實作
    以 Python 為核心，介紹地理與環境資料之程式化處理流程， 包含 CSV、JSON 與 Shapefile（SHP）等資料格式之讀取與轉換， 並以 Pandas、GeoPandas 進行向量資料處理， 搭配 Rasterio 分析遙測影像與 DEM，完成整合式實作分析。
  • 環境教育與災害防救
    依據環境教育人員資格（災害防救、自然保育）， 規劃環境教育與防災教育相關課程， 結合實際災害案例、環境監測資料與空間分析成果， 適合通識課程、環境教育或跨域學程開設。
  • 中學地理教學與空間思維導入
    結合中等教師證（國中社會學習領域、高中地理）之專業背景， 探討地理教育中之空間思維培養， 並示範如何將 GIS、遙測與環境議題融入中學與通識教學。
  • 無人機空拍基礎與應用實務
    依據無人機專業基本級操作證（2 公斤以下）， 介紹無人機操作規範、飛航安全與任務規劃， 並涵蓋空拍影像蒐集、正射影像與 DEM 建立流程， 進一步說明 UAV 影像於環境監測與災害調查中的 影像辨識與 AI 輔助分析應用（如 YOLO）。
